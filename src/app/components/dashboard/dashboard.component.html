Given a number n as an input. Find the sum of the nth Fibonacci number and nth Prime number. Prime numbers are 2, 3, 5, 7, 11, 13.... Fibonacci numbers are 1, 1, 2, 3, 5, 8, 13... (starting from 1).

Input Format

Input contains single integer n

Constraints

1 <= n <= 40

Output Format

Output should be single integer representing the sum.

Sample Input 0

3

Sample Output 0

7

Explanation 0

3rd prime is 5 and 3rd Fibonacci is 2. So sum is 5+2=7

Sample Input 1

6

Sample Output 1

21

Explanation 1

6th prime is 13 and 6th Fibonacci is 8. So sum is 13+8=21.
